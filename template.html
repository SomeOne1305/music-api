Welcome to ♫ MUSIC API

It is free to use this API. I have used it. I find this very helpful for everyone especially for junior developers, so I share this for you ! ☺

Get users list who are registered:

Get https://free-music-source.vercel.app/api/v1/users
GET /api/v1/users
GET /api/v1/users/{:index}

Get musics for using:

Get https://free-music-source.vercel.app/api/v1/musics
GET /api/v1/musics
GET /api/v1/musics/{:index}
